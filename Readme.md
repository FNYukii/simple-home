# Simple Home
## 概要
「インテリアデザイン会社のホームページ」をテーマに制作したWebサイト。

[Simple House](https://github.com/Yu357/SimpleHouse)を`React`+`JavaScript`で再実装したものになります。

## 制作時期
2022年5月

## 使用したモジュール
- `react-router-dom`
- `react-icons`
- `firebase`
