# Simple Home
## 概要
Reactの勉強として制作したWebサイト。
インテリアデザイン企業のホームページというテーマで作成しました。
画面のデザインは[rustafied.com](https://www.rustafied.com/)を参考にさせていただきました。

## 使用したモジュール
- react-router-dom
- react-icons
- firebase
